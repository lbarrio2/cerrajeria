# Cerrajería Newen - Página Web Profesional

Página web moderna y profesional para servicios de cerrajería a domicilio y automotriz.

## 🎯 Características

- ✨ **Diseño Responsivo** - Se adapta a todos los dispositivos (móvil, tablet, desktop)
- 🎨 **Interfaz Moderna** - Colores profesionales y animaciones suaves
- 📱 **Menú Hamburguesa** - Navegación optimizada para móviles
- 🖼️ **Galería Interactiva** - Galería de trabajos con overlay animado
- 📧 **Formulario de Contacto** - Con validación completa
- ⚡ **Rendimiento** - Carga rápida y optimizado
- 🔒 **SEO Friendly** - Estructura semántica HTML5

## 📂 Estructura de Archivos

```
├── index.html      # Página principal HTML
├── styles.css      # Estilos CSS
├── script.js       # Funcionalidad JavaScript
└── README.md       # Este archivo
```

## 🚀 Cómo Usar

### 1. Clonar o Descargar
```bash
git clone https://github.com/lbarrio2/cerrajeria.git
cd cerrajeria
```

### 2. Abrir en el Navegador
Simplemente abre el archivo `index.html` en tu navegador web.

## 🎨 Personalización

### Cambiar Datos de Contacto

Abre `index.html` y busca:
- **Teléfono**: `+56 9 0000 0000` (línea ~122)
- **Email**: `info@cerrajerianewen.cl` (línea ~128)
- **Ubicación**: `Chile` (línea ~134)

### Cambiar Colores

Abre `styles.css` y modifica las variables en `:root`:

```css
:root {
    --primary-color: #d4af37;    /* Color dorado */
    --secondary-color: #1a1a1a;  /* Color negro */
    --tertiary-color: #2a2a2a;   /* Color gris oscuro */
    --text-color: #333;          /* Color del texto */
    --light-bg: #f5f5f5;         /* Fondo claro */
    --white: #fff;               /* Blanco */
}
```

### Agregar Imágenes Reales

En `index.html`, reemplaza las URLs de imágenes placeholder:

```html
<!-- Cambiar de: -->
<img src="https://via.placeholder.com/300x250?text=..." alt="...">

<!-- A: -->
<img src="tu-imagen-aqui.jpg" alt="...">
```

### Modificar Servicios

Los servicios están en la sección `<section class="servicios">`. Puedes:
- Cambiar títulos
- Actualizar descripciones
- Agregar o quitar items de la lista

## 🔧 Funcionalidades JavaScript

- **Menú Hamburguesa**: Toggle responsive del menú
- **Scroll Suave**: Navegación fluida entre secciones
- **Validación de Formulario**: Validación de campos obligatorios
- **Animaciones de Scroll**: Efectos al hacer scroll
- **Sección Activa**: Marca la sección actual en el menú
